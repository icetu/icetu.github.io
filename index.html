What is Icetu Linux ?

Icetu Linux is a Ubuntu-based distribution. Because Icetu Linux installs many additional packages. This distribution allows the user to install its own packages and save system resources than Ubuntu. Icetu Linux also uses the Pacman Package Manager & Linux Kernel LTS.

Icetu Base

Icetu Base is a minimal rootfs for use in the creation of custom images for specific needs. Icetu Base strives to create a suitable minimal environment for use in Board Support Packages, constrained or integrated environments, as the basis for application demonstration images. Icetu Base is available for the amd64 architectures.
